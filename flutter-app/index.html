How to Setup Flutter in Android Studio

  1. Download Lastest Flutter SDK from below link.in that click on flutter_window_xxx.zip button.
  2. Flutter sdk Here



  3. Extract the zip file and copy flutter folder into your desired installation location for the Flutter SDK (eg. C:\src\flutter; do not install Flutter in a directory like C:\Program Files\).

  4. Inside Flutter folder find flutter_console.bat. Start it by double-clicking.<\li>


  5. Now set your enviournment variable
  6. From the Start search bar in , type ‘env’ and select Edit environment variables for your account



  7. Under User variables check if there is an entry called Path:
  8. Click on Edit.new Dialog open from it click on new and copy full path to flutter\bin as its value
  9. Now Restart your pc for changes to take effect


  10. Now check everything is ok or not
  11. open cmd and do following shown in below picture



  12. Now open Android Studio and create new flutter project then select your flutter sdk file to the location where we have downloaded earlier and that's it




How to Change app logo

  1. For Android, open android > app > src > main > res > mipmap add here your logo according to device screen size



  2. For iOS, open ios > Runner > Assets.xcassets > AppIcon.appiconset here you need to put your logo according to diffrent size.

How to Change App Colors

You can dynamically change the app colors from the admin panel. other colors will dynamically generated from primary and background color.

And as a fallback you can add the primary and background colors in the app code. when there is some issue loading colors from admin panel these colors will be used.

How to Setup Admob

  1. Open lib/utility/constants.dart, then you can add your Banner & Interstitial Ad Ids for each platform as shown in the image.
  2. After that, you will need to add your ad app id in for Android by going to android/app/src/main/AndroidManifest.xml
  3. And for IOS, by going to ios/Runner/Info.plist

How to enable support for RTL in app

You can change the directionality of the app for RTL languages, as per your requirement by going to lib/utility/constants.dart



Generate Release APK

  1. In you android studio, right click on you android module of you flutter project.there is option flutter, click on that inside open in new module.then generate release apk from that new module.



  2. and if in your android studio if flutter option is disable then go to file option in your android studio inside open then locate your flutter project's android module and open it.now generate relase apk from that new module.





  3. In Build menu there is option generate signed bundle/apk.
